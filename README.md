# Tip Calculator – Android Jetpack Compose

Une application Android simple et moderne permettant de calculer le pourboire en fonction du montant de la facture, du pourcentage choisi, et de l’option d’arrondi.  
Ce projet a été développé en **Kotlin** avec **Jetpack Compose** et **Material 3** pour une interface élégante et intuitive.

---

##  Fonctionnalités

-  Calcul automatique du pourboire selon un pourcentage saisi  
-  Option pour **arrondir** le montant du pourboire  
-  Entrée fluide du montant et du pourcentage grâce aux **TextFields** personnalisés  
-  Interface construite avec **Jetpack Compose**  
-  Support du thème **Material 3**  
-  Design responsive et ergonomique  


##  Aperçu du fonctionnement

L’application permet de :
1. Saisir le **montant de la facture**   
2. Saisir le **pourcentage du pourboire**   
3. Activer/désactiver l’option pour **arrondir le pourboire**   
4. Voir instantanément le **montant du pourboire calculé**
